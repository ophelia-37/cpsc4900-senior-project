# Sample Images

This directory is for storing test artwork images for the restoration application.

## Where to Get Sample Images

### Public Domain Art Collections

All of these sources offer high-resolution images of artworks that are in the public domain:

#### 1. Metropolitan Museum of Art
- **Website**: https://www.metmuseum.org/art/collection/search
- **Search Tips**: 
  - Use the "Open Access" filter
  - Download high-resolution images
  - Great for paintings from all eras

#### 2. Rijksmuseum (Netherlands)
- **Website**: https://www.rijksmuseum.nl/en/rijksstudio
- **Search Tips**:
  - All images are public domain
  - Excellent Dutch masters collection
  - Very high resolution available

#### 3. National Gallery of Art (Washington, DC)
- **Website**: https://www.nga.gov/open-access-images.html
- **Search Tips**:
  - "Open Access" images are free to use
  - American and European art
  - Good variety of styles

#### 4. The Getty Museum
- **Website**: https://www.getty.edu/art/collection/
- **Search Tips**:
  - Filter by "Open Content"
  - Excellent for photographs and paintings
  - High-quality scans

#### 5. Wikimedia Commons
- **Website**: https://commons.wikimedia.org/
- **Search Tips**:
  - Search for "paintings" or specific artists
  - Check license (look for Public Domain)
  - Variable quality

## Recommended Test Cases

### For Inpainting Testing

1. **Portraits**: Test on faces, skin tones
2. **Landscapes**: Test on natural textures (trees, water, sky)
3. **Architecture**: Test on structured elements (buildings, columns)
4. **Abstract**: Test on patterns and textures

### For Color Correction Testing

1. **Faded paintings**: Older artworks with color degradation
2. **Dark paintings**: Underexposed or darkened with age
3. **Color cast**: Paintings with yellow/blue tints
4. **High contrast**: Paintings with very light and dark areas

## Sample Image Suggestions

### Easy Difficulty
- Simple portraits with plain backgrounds
- Landscapes with large uniform areas
- Still life paintings

### Medium Difficulty
- Portraits with detailed backgrounds
- Complex landscapes
- Interior scenes

### Hard Difficulty
- Group portraits with many faces
- Highly detailed architectural scenes
- Paintings with intricate patterns

## Image Specifications

For best results with the restoration tool:

- **Format**: JPG, PNG, or TIFF
- **Resolution**: 1024-2048 pixels (largest dimension)
- **File Size**: Under 10 MB
- **Color Space**: RGB
- **Bit Depth**: 8-bit per channel

## Downloading Instructions

### From Met Museum:
1. Find an artwork you like
2. Click "Download" button
3. Choose "Large" size (typically 2000+ pixels)
4. Save to this directory

### From Rijksmuseum:
1. Search for an artwork
2. Click on the image
3. Click "Download" icon (down arrow)
4. Choose highest resolution
5. Save to this directory

### From Getty:
1. Search for an artwork
2. Click "Download Image"
3. Select resolution
4. Accept terms and download
5. Save to this directory

## Organizing Your Images

Suggested naming convention:
```
artist_title_year.jpg

Examples:
- vangogh_starrynight_1889.jpg
- rembrandt_nightwatch_1642.jpg
- monet_impression_1872.jpg
```

## Creating Synthetic Test Cases

You can also create synthetic damaged images:

1. Take any image
2. Use an image editor (GIMP, Photoshop, etc.)
3. Add artificial damage:
   - White/black scratches
   - Missing regions
   - Faded areas
4. Save both damaged and original versions

## Testing Workflow

1. **Start Simple**: Begin with easy images to learn the tools
2. **Increase Difficulty**: Gradually try more complex artworks
3. **Document Results**: Keep notes on what works well
4. **Compare Techniques**: Try different algorithms on the same image
5. **Build Portfolio**: Save your best restorations for demonstration
